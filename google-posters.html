AI* Poster Series

Designing a system, then letting AI execute it.

*Claude Code was used to generate 16 poster designs in 2 days, from a templatized HTML design system I architected. This experiment tested computer vision interpretation and the limits of stylistic variation within a constrained design system.

THE EXPERIMENT

I wanted to understand how far I could push AI as a creative executor while keeping the output within a coherent design system. The experiment: architect a templatized HTML design system based on traditional type specimen structure, feed Claude Code a set of image assets, and use computer vision to interpret and translate them into a series of digitized posters.

THE CONSTRAINT

The constraint was the point. By defining the system first—grid, typographic hierarchy, layout logic—the AI had a rulebook to work within and a boundary to push against. The result was 16 distinct poster designs generated in two days, each visually differentiated but systemically coherent.

WHAT IT PROVED

Good AI output is downstream of good design thinking. The system I built was the real artifact. The posters were the proof.
